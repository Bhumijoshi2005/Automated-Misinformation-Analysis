📰 Automated Misinformation Analysis
Automated Misinformation Analysis is designed to identify misinformation in textual content by applying NLP and Machine Learning techniques. The system preprocesses user input, extracts meaningful features, and predicts whether the content is genuine or misleading using a trained Logistic Regression model.

✨ Features

🔍 Automated misinformation detection
🧹 Text preprocessing and cleaning
🚫 Stopword removal using NLTK
🌱 Stemming using Porter Stemmer
📊 Feature extraction using CountVectorizer
🤖 Logistic Regression based classification
⚡ Fast and efficient prediction system
📈 Model evaluation and performance analysis
